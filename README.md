## Overview

This repository is a bootcamp training guide to help you pass the **Databricks Preferred Partner (DPP)** certification. There are READMEs throughout this repo with detailed information about each section. At it's heart, this repo contains practice datasets to help you practice for the DPP certification. ## What is the DPP certification?

The Databricks Preferred Partner (DPP) certification is a professional certification program that validates expertise in using Databricks for data science and machine learning. The certification process involves 2 steps:

1. **Interview 1 - Live Data Science Coding** (1 hour): A hands-on coding interview where candidates work through a data science problem in the Databricks platform. This includes:
   - Exploratory Data Analysis (EDA) on dirty, real-world datasets
   - Data cleaning and preprocessing
   - Building production-ready machine learning models
   - Handling common data quality issues (missing values, class imbalance, outliers, etc.)

2. **Interview 2 - Generative AI Discussion** (30 minutes): A conversation about GenAI projects, evaluation methods, and solution architecture. This includes:
    - Talking about past Gen AI projects you've done
    - Gen AI best practices and frameworks

The certification demonstrates proficiency in:
- Working with messy, production-like datasets
- Building robust, production-ready ML pipelines
- Using Databricks platform features effectively
- Applying best practices in data science workflows
- Gen AI Fundamentals

For complete certification details and preparation guidance, see:
- [Interview 1 - Machine Learning Coding](documents/INTERVIEW_1_MACHINE_LEARNING.md) for Interview 1
- [Interview 2 - Generative AI Discussion](documents/INTERVIEW_2_GENAI.md) for Interview 2

## DPP Preparation Steps

Please follow these steps to complete the bootcamp.
- **Step 1**: Read the interview guides:
  - These 2 guides contain essential information about how the certification works. It is essential that you understand how the exam works so that you can prepare accordingly, so please first read both of these documents.
  - [Interview 1 - Machine Learning Coding](documents/INTERVIEW_1_MACHINE_LEARNING.md)
  - [Interview 2 - Generative AI Discussion](documents/INTERVIEW_2_GENAI.md)
- **Step 2**: Draft out an Plan of Attack
  - You need to have a well established plan before you enter the proctored 1-hour coding exam. This plan should be a simple bulleted list of tasks (maybe 8 to 15) that you are going to do in the exam regardless of the dataset they give you. You need to be able to memorize these steps for the exam, and you only have 1 hour for the exam, so be strategic about the steps you choose.
- **Step 3**: Choose a dataset and begin testing your framework
  - Now you need to practice in a relaxed setting (not timed). Take your Plan of Attack and begin creating prompts for each step that you will execute with the Databricks AI Assistant. You are effectively practicing using the Databricks platform because you HAVE to use the Databricks AI Assistant to finish the exam in time. You won't be able to bring your prompts with you to the exam, so the more you practice working the with Databricks AI Assistant, the better you'll get at this.
  - Practice with a few datasets. Maybe 5 or so (save a couple datasets for step 4).
- **Step 4**: Timed Practice Testing using the datasets
  - Once you think you're ready, make an appointment with whoever is leading you through this bootcamp (e.g., Mike, Steve) and actually have a timed coding assignment. 
  - Use a dataset you've never seen before.
- **Step 5**: Take the exam for real
  - You've done all the prep, now let's go for it!
